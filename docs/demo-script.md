# Demo Walkthrough

## 1. Dashboard

Start on the dashboard and review the high-level health cards.

Point out:

- failed jobs
- stale jobs
- open incidents
- pending validation checks
- recent health summary results

## 2. Incidents

Open the Incidents page and review the incident register.

Create a new incident to show the workflow.

## 3. Incident Details

Open an incident detail page.

Add a timeline note, then resolve the incident with a root cause and resolution summary.

## 4. Audit Log

Open the Audit Log page and confirm the incident actions were recorded.

## 5. Exports/API

Open the Exports/API page.

Show the JSON endpoints and CSV export links.

## 6. Tests

Run dotnet test from the repository root.
